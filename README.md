# MERN Stack Final Year Project

This is my Final Year Project developed using MERN Stack.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Features
- User Registration & Login
- CRUD Operations
- Role Based Access (if any)
- Responsive UI

## How To Run

Backend:
cd server
npm install
npm start

Frontend:
cd client
npm install
npm start
