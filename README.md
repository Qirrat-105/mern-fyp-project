# MERN Stack Final Year Project

This is my Final Year Project developed using MERN Stack.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Features
- User Authentication (Login/Register)
- Admin Dashboard
- CRUD Operations
- Responsive UI

## How to Run Project

1. Download client.zip and server.zip
2. Extract both folders

Backend:
cd server
npm install
npm start

Frontend:
cd client
npm install
npm start

## Environment Variables
Create .env file inside server folder and add:

MONGO_URI=your_mongodb_connection
PORT=5000
